# Taller 1 – MySQL con Node.js

Este proyecto contiene el desarrollo del Taller 1 usando MySQL, Node.js y consultas básicas desde JavaScript.

## Tecnologías utilizadas

Node.js

Express

MySQL

mysql2

JavaScript

## Contenido del proyecto

index.js — Servidor Express y endpoints.

database.js — Conexión a MySQL.

firebase_queries.js — Operaciones de consulta (CRUD).

taller1.sql — Script para crear BD y tablas.

## Cómo ejecutar el proyecto

Instala las dependencias:

npm install


Configura tu base de datos en database.js.

Ejecuta el servidor:

node index.js

## Base de datos

Ejecuta el archivo taller1.sql para crear las tablas necesarias.

## Notas

El proyecto requiere un servidor MySQL activo.

Las consultas están implementadas usando mysql2/promise.
